# 發出指令

目標：使用 Swift 指令，讓 Byte 動起來，去收集寶石。

## 介紹

你的角色 Byte 喜歡收集保石，但他無法獨自做到，在第一的關卡中，你需要編寫 Swift 指令，讓 Byte 在他的世界中到處移動來搜尋寶石。

1. 尋找關卡世界的寶石
2. 輸入正確的 `moveForward()` 和 `collectGem()` 指令組合
3. 按下「執行我的程式碼」
