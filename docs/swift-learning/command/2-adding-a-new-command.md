# 加入新指令

目標：把移動和轉彎這兩個指令組合起來，去收集寶石。

## 簡介

這一關和上一關類似，但這次 Byte 需要左轉才能到達寶石處。你可以使用上一關的指令以及新指令：`turnLeft()`。

1. 構思讓 Byte 移動到寶石處並收集寶石所需的指令。
2. 在程式碼區域輸入指令。
3. 執行程式碼。
